Initializing Riot

This demonstration shows how to setup your base page with RiotJS. In this example (see code) I created two custom RiotJS components found in the tags folder. Riot tags look like HTML/CSS/JS combined. While it takes on the standard format of HTML/CSS/JS, it is not recognized as any. It is its own unique thing. What RiotJS does, is allow us to write these kinds of tag files, then compile them into pure JS for the browser to understand.
